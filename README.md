# handshaking
Tremor sensing

## install venv
```
python -m venv venv
```

## activate venv
on mac and linux
```
source/bin/activate
```

on windows CMD
```
.\venv\Scripts\activate.bat
```
on windows powershell
```
\venv\Scripts\activate.ps1
```

## install requirements

```
pip install -r requirements.txt
```
